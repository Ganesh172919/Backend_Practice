# Roadmap

## Purpose

This document lists realistic future directions for the project.

## Short-Term Roadmap

- add checkpoint save/load support
- add automated tests
- add a quieter `--quiet` CLI mode
- improve report visual polish

## Medium-Term Roadmap

- add automated interaction detection
- add stronger tabular baselines
- add more systematic hyperparameter sweeps
- add richer explanation diagnostics

## Long-Term Roadmap

- support higher-order additive interactions
- support deployment-oriented inference APIs
- support broader paper extensions beyond tabular data

## Contribution Guidance

If you want to work on one thing that will most improve the project, start with:

1. tests
2. checkpointing
3. interaction selection
4. benchmark tuning
