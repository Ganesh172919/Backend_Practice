# Glossary

## Additive Model

A model that represents the prediction as a sum of interpretable component functions.

## GAM-1

A generalized additive model with only univariate feature functions.

## GAM-2

A generalized additive model that includes univariate and pairwise interaction functions.

## Functional ANOVA

A decomposition of a function into components associated with subsets of variables.

## SHAP / Shapley Value

A feature attribution framework that distributes model output across input features based on coalition behavior.

## Shapley Kernel

A weighting scheme over subsets used in least-squares-style Shapley formulations.

## Masked Surrogate

A model trained to approximate the output of a black-box model under masked feature subsets.

## InstaSHAP

An additive model trained with a masked objective so that SHAP-like attributions can be extracted instantly.

## Synergistic Interaction

An interaction where the joint information of features creates additional useful signal.

## Redundant Interaction

An interaction setting where correlated features overlap in the information they provide, which can distort simple additive explanations.
