# File Reference: `requirements.txt`

## Purpose

`requirements.txt` lists the Python dependencies needed to run the project.

## Important Libraries

- `torch`: neural model training
- `numpy`, `pandas`: numerical and tabular processing
- `scikit-learn`: preprocessing, metrics, optional tree baseline
- `shap`: SHAP baseline explanations
- `ucimlrepo`: dataset loading
- `matplotlib`, `seaborn`: plotting
- `PyYAML`: configuration loading
- `tensorboard`: training log inspection

## Why It Matters

If the project fails due to import errors or environment mismatch, start here.
