# File Reference: `utils/visualization.py`

## Purpose

This file centralizes all plot generation logic.

## Main Plot Types

- training curves
- metric bar charts
- shape functions
- interaction heatmaps
- feature importance plots
- SHAP vs InstaSHAP alignment plots

## Why It Matters

This file converts model internals and saved results into interpretable artifacts. It is the main place to modify the visual style or add new plot types.
